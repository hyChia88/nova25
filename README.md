# CheatSheet - PDF Summary Generator

使用 OpenRouter API 的 PDF 文件分析和摘要生成工具

## 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python app.py
```

### 3. 访问应用
打开浏览器访问：http://localhost:5000

## 功能特点

- 📤 支持拖放和点击上传 PDF 文件
- 🤖 使用 OpenRouter API (GPT-4) 生成智能摘要
- 🎨 极简主义设计，浅色主题
- ⚡ 实时处理和反馈
