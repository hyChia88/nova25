# CheatSheet - PDF Concept Extractor

使用 OpenRouter API 的 PDF 文件概念提取工具

## 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 运行应用
```bash
python app.py
```

### 3. 访问应用
打开浏览器访问：http://localhost:5000

## 功能特点

- 📤 支持拖放和点击上传 PDF 文件
- 🤖 使用 OpenRouter API (GPT-4) 自动提取关键概念
- 📋 结构化 JSON 输出，每个概念包含标题和描述
- 💾 自动保存结果到 `pdf2points_example.json` 文件
- 🎨 极简主义设计，浅色主题
- 💡 概念卡片式展示，清晰直观
- ⚡ 实时处理和反馈
